URL Encode: Percent-encodes special characters for use in URLs. Optionally encode slashes or use + for spaces.
Unicode Escape: Converts characters to \uXXXX sequences and back.
Base64: Standard or URL-safe Base64 encoding/decoding.
HTML Entities: Encodes & < > " and all non-ASCII as named (e.g. &eacute;, &mdash;) or numeric entities. Use named entities for C# XML doc comments and markdown.
Hex: Converts each byte to its hexadecimal representation with configurable separators.
ROT13: Classic letter rotation cipher (encode and decode are the same operation).
