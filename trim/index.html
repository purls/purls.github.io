Character Selector

Controls:
• Click a box to toggle that position on/off
• Ctrl+Click to enter arrow mode, then press ← to delete that position and everything before, or → to delete that position and everything after
• Click [...] to toggle keeping/removing characters beyond the shortest line
• Press Esc to exit arrow mode

Range Expression

Format: {index} for single character, {start:end} for range (exclusive end), {start:} for start to end of line.
Example: {1}{4:7} keeps character at index 1 and characters from index 4 to 6.

Trimmed Output

Use this tool to extract specific columns or character positions from multi-line text. Useful for working with fixed-width data, logs, or any text where you need to extract consistent positions across many lines.

Tip: The character selector shows the shortest line as a reference. Characters beyond the shortest line can be toggled with the [...] button.
