Build a chain of text operations that execute in order. Each operation's output feeds into the next. Use Per Line mode to process each line individually, or Whole String to treat the input as one block. Every operation supports optional conditions (if contains, if length >, etc.) to selectively apply. Press Ctrl+Enter to run the pipeline. Use ⇄ Swap to feed the output back as input. Export/Import lets you save and reload pipeline configurations as JSON.
